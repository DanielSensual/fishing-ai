# Gemini Demo App

A minimal SwiftUI app that calls Google's Gemini model via the Generative Language API.

## Setup

1. Copy `Secrets.example.plist` to `Secrets.plist` and set your key:

   - Open Terminal and run:
   - Edit `Secrets.plist` and replace `YOUR_API_KEY_HERE` with your actual API key.
   - In Xcode, select `Secrets.plist`, open the File Inspector, and ensure your app target is checked under Target Membership so it’s bundled in the app.
